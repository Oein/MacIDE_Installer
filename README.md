# Mac IDE Installer for Oein

```sh
/bin/bash -c "$(curl -fsSL https://github.com/Oein/MacIDE_Installer/raw/main/macinstaller.zsh)";
```

# What kind of things does it install?

- Brew
- NVM
- Node 16
- Oh My Zsh
- wget
- p10k
- iTerm
- Spotify
- Chrome
- Visual Studio Code
- Sublime Text
- Keka
- Discord
